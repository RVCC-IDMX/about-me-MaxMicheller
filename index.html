Embedded static Google map using iFrame

Embedded static Google map using iFrame sounded like an interesting topic to look into when choosing one to research. To research about it, using google, would be the first step. Finding the documentation for the google maps api which lead me to api keys, but since that isnt what is needed. Just a static map will do. Which was easy enough to do. Just open Google maps, find the location you want to use. Go to share, then embed a map, copy the html and copy that to your website. There are many positive ways you can utalize this, like if you had a website for a small buisness you own. Having a static embed will help people easily find the buisness.
